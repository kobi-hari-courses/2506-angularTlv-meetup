About This Demo

This app demonstrates a modern, themable Angular chat interface for support and sales teams. It features separate chatbots for each department, a fully responsive layout, and theme colors powered by CSS custom properties.
The dashboard shows live (simulated) chat activity so you can track engagement across your teams.
